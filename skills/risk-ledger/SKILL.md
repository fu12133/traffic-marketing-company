## 风控评分模型 v2.0

```
风控评分 = 基础分100 - 扣分项 + 加分项 + 行业系数调整

行业系数:
  医美/医疗: ×1.2 (更严格)
  金融:      ×1.15
  教育:      ×1.1
  电商:      ×1.0 (基准)
  游戏:      ×0.95
  其他:      ×1.0
```

### 扣分项 v2.0

| 违规类型 | 单次扣分 | 说明 |
|---|---|---|
| 一级违规（🔴）封号 | -30 | 政治敏感/色情/虚假医疗/金融欺诈 |
| 二级违规（🟠）封禁 | -20 | 行业敏感词/功效承诺/资质缺失/屡教不改 |
| 三级违规（🟡）警告 | -10 | 绝对化用语/数据无出处/对比贬低 |
| 四级违规（🔵）限流 | -5 | 擦边词/不规范表述/建议优化 |
| 重复违规 | 额外 -5/次 | 同一广告主30天内同类违规 |
| 超期未整改 | -15 | 超过整改期限48h未完成 |
| 申诉被驳回 | -10 | 违规申诉不成立 |
| 用户投诉核实 | -15 | 经核实的用户投诉（多人叠加） |
| 平台处罚同步 | -20 | 被投放平台处罚同步到台账 |
| 资质过期 | -10 | 行业资质未及时续期 |

### 加分项 v2.0

| 行为 | 加分 | 说明 |
|---|---|---|
| 连续30天无违规 | +5 | 每个30天周期，上限+20 |
| 主动报送风险 | +3/次 | 广告主自查发现风险并报告，月上限3次 |
| 超快速整改（<4h） | +2/次 | 违规后快速完成整改，月上限4次 |
| 通过合规培训 | +10 | 参加合规培训并通过考核，年上限1次 |
| 获得合规认证 | +15 | 通过行业合规认证（如ISO认证） |
| 提供优质素材 | +1/百条 | 百条素材无违规的稳定性奖励 |

### 风险等级划分 v2.0

| 评分区间 | 风险等级 | 标识 | 管理措施 | 抽检比例 |
|---|---|---|---|---|
| 90-100 | 低风险 | 🟢 | 正常投放，自动化审核 | 5% |
| 70-89 | 中低风险 | 🔵 | 正常投放，加大抽检 | 15% |
| 50-69 | 中风险 | 🟡 | 观察名单，人工抽审 | 30% |
| 30-49 | 高风险 | 🟠 | 限制投放，全量人工审核 | 50% |
| <30 | 黑名单 | 🔴 | 暂停合作，清退处理 | 100% |
| 一票否决 | 黑名单 | 🔴 | 直接清退，不可逆 | - |

## 规则库更新机制

### 更新触发条件

```
┌─ 定时更新
├─ 每日: 增量更新（平台政策微调）
├─ 每周: 合规团队评审更新
├─ 每月: 全量规则库版本迭代
└─ 季度: 重大法规适应性更新

┌─ 事件触发
├─ 平台政策重大变更 → 48h内紧急更新
├─ 新法规发布 → 72h内完成适配
├─ 新行业入驻 → 上线前完成规则配置
├─ 新型违规出现 → 24h内新增检测规则
└─ 申诉案例积累 → 每周案例评审后更新
```

### 更新流程

```
Step 1: 信息收集
  ├─ 平台官方公告监测（抖音/微信/小红书/快手/Meta/Google）
  ├─ 法规政策跟踪（广告法/网信办/市场监管总局）
  ├─ 违规案例库分析（内部历史数据 + 行业公开案例）
  ├─ 审核团队反馈（审核中遇到的新问题）
  └─ 广告主申诉分析（判定边界案例）

Step 2: 影响评估
  ├─ 新增规则覆盖范围
  ├─ 修改规则影响面
  ├─ 废弃规则副作用
  ├─ 规则敏感度测试（避免过度拦截）
  └─ 误报率影响评估

Step 3: 规则开发
  ├─ 新增违禁关键词
  ├─ 修改检测规则逻辑
  ├─ 调整风险等级映射
  └─ 更新平台处罚对照表

Step 4: 内测验证
  ├─ 使用历史10%样本测试
  ├─ 计算召回率 vs 精确率
  ├─ 对比旧版规则差异
  └─ 收集内测问题反馈

Step 5: 灰度发布
  ├─ 选择5%流量灰度
  ├─ 监控7天效果
  ├─ 收集误报和漏报
  └─ 确认无误后全量发布

Step 6: 全量上线
  ├─ 版本号更新
  ├─ 更新日志发布
  ├─ 审核团队培训通知
  └─ 广告主合规提醒（重大变更）
```

### 规则版本号规范

```
格式: v{主版本}.{次版本}.{修订号}-{环境}

主版本: 重大法规/平台政策变更（如新广告法）
次版本: 规则分类新增/修改（如新增医美类规则）
修订号: 关键词增删/阈值调整（如添加新违禁词）

示例:
  v3.2.1-stable   → 当前生产版本
  v3.2.2-beta     → 内测中的下一个版本
  v4.0.0-alpha    → 重大改版开发中
```

| 行业日违规率 > 20% | 全行业素材全量人工审核 | 审核主管 + 业务 |
| 同一违规类型周环比 +50% | 发送专项提醒至受影响广告主 | 广告主运营 |
| 广告主评分降至 < 30 | 自动进入清退流程 | 风控主管 + 法务 |
